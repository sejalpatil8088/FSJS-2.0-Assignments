HTML

HTML stands for Hyper Text Markup Language. Html is the standard markup language for creating web pages.

Html decribes the structure of a web page.Html consists of a series of elements.Html elements tell brower how to display the content.

Web

The web receives the request,finds the resources and return the response to the client.

The part of the internet that can be looked at with a special program and that is made up of any documents which are linked together.

Servers:-

live server

Makes our existing server live.This is a web extension that helps to live reload feature for dynamic content. This is a easy way to run our program.

In the context of websites and web servers , a devloper will often put a copy of a website on a staging server first.Acces to a website on staging server is usually restricted so that only devlopment team and the client can see the site .Once the client is happy wuth the site its then deployed to a live server.

Apache server

Apache server is a free and open source web server that delivers web content through the internet. A client connects to a server with the specified protocol and makes a request for a resource using the url-path
