Positions

The position css property sets how an element is positioned in a document.

After applying position we can apply top,right,bottom,and left properties determine the final location of positioned elements. The position property specifies the type of positioning method used foe an element(relative,fixed,absolute or sticky).

Relative:-

Relative will calculated from top,bottom,right,left from its orignal area.

Absolute:-

Absolute property is calculate from its parent area.

Fixed:-

An element with position: fixed; is positioned relative to the viewport ,which means it always stays in the same place even if the page is scrolled .

Sticky:-

An element with position:sticky; is positioned based on the user's scroll position.
